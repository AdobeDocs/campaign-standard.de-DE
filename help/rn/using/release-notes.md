## Version 26.2 {#26.2}

### Korrekturen von Sicherheitsproblemen {#26-2-security}

* Diese Version enthält mehrere Korrekturen von Sicherheitsproblemen.
* Diese Version enthält die folgenden Sicherheitsaktualisierungen: Debian 13 und PostgreSQL 17.

### Andere Fehlerbehebungen {#26-2-fixes}

* Optimierung dynamischer Berichte zum Erhöhen der Stabilität und Verringern von Datendiskrepanzen. (CAMP-58964)
* Verbesserter IMS-Anmeldemechanismus für mehr Zuverlässigkeit. (CAMP-59418)
* Verbesserte AEM-Inhaltsintegration mit besserer URL-Verarbeitung für paginierte Anfragen. (CAMP-59540)
* Unterstützung für die Eingabe des Kennworttyps wurde im Bildschirm „Optionen“ mit automatischem Hashing hinzugefügt. (CAMP-60206)
* Es wurde ein Problem behoben, bei dem temporäre Workflows falsche Sequenz-IDs verwendeten. (CAMP-59824)
* Es wurde ein Problem behoben, das zu Fehlern beim Abrufen von AEM-Kampagneninhalten führte. (CAMP-59926)
